== Description ==

Provides easy way to 'roll back' and 'roll forward' plugin versions that support version tagging.

Please note that this plugin cannot reverse database upgrade routines that have been implemented. You will want to contact and ask the plugin developer if it is safe to roll back a version.

= Highlights =

* Roll back plugin versions for plugins that support version tags.
* Roll forward plugin versions for plugins that support version tags.

== Installation ==

1. Upload `version-control` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

1. How do I use this tool?

Open up your plugins listing page and if your plugin supports version tags then a version selection with switch button will be available
